# tt

A task management command tool integrated with AI

## Usage

```sh
tt [-v | --version] [-h | --help] <subcommand> [<args>]
```

## Flags

- `-v`, `--version`
    - バージョンを出力します
- `-h`, `--help`
    - ヘルプを出力します
